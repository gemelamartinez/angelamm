Tipos de selectores que entran en examen

A continuación se van a explicar los siguientes selectores:

  1. Clase (class=””): selector que sirve para cambiar el color de la letra o del fondo de los elementos del texto que se quieran cambiar .Si se puede reptir su nombre . Véase el ejemplo: Vamos a cambiar el color letra a morado
  2. Identificador (id=””): selector que sirve para cambiar el color de la letra o del fondo de los elementos del texto que se quieran cambiar . No se puede reptir su nombre. Ejemplo: Vamos a cambiar el color del texto de "h1" a verde.
  3. Todos los elementos (*): selector general utilizado para cambiar el fondo y el color predeterminados.Ejemplo: Todo el fondo es azul .
  4. Varios elementos ( p, em, h1, h2): selector utilizado para cambiar el fondo y el color predeterminados de todo el texto de los elementos que están entre paréntesis. Ejemplo: Vamos a cambiar el fondo a naranja.
  5. Elementos contenidos en otros elementos (ol li):lector utilizado para cambiar el fondo y el color predeterminados del texto de los elementos que están dentro de un selector. En este caso sería se cambiarían los "li" de los "ol". Ejmplo:Vamos a cambiar el color de la letra de los "li" a rosa.
